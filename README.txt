Portfolio Website
About the Project
This is my personal Portfolio Website built using HTML, CSS, and JavaScript. It serves as a digital showcase of my skills, projects, and experience in web development. The website features a clean and modern design, making it easy for visitors to explore my work and connect with me.

Features
Fully Responsive Design – Works on all devices, including desktops, tablets, and mobile phones.
Smooth Animations & Transitions – Enhances user experience with visually appealing effects.
Projects Showcase – Displays my web development projects with live links and descriptions.
Contact Form – Allows visitors to get in touch with me easily.
Optimized Performance – Ensures fast loading times and efficient code.
Technologies Used
HTML5 – For structuring the webpage content.
CSS3 – For styling, animations, and responsive layout.
JavaScript – For interactive elements and dynamic content.
